# Botball 器材检查器 2026

🤖 一个用于 Botball 比赛的零件使用跟踪工具，帮助队伍管理两个机器人的零件使用情况，确保符合比赛规则。

## ✨ 功能特性

### 📊 零件管理
- **三大套件**：电子套件、金属零件、乐高零件
- **双机器人跟踪**：分别记录两个机器人的零件使用
- **实时统计**：总零件类型、已使用类型、超规零件
- **超规警告**：超过限制的零件会红色高亮显示

### 🎯 用户体验
- **快速输入**：加减按钮快速调整数量
- **搜索功能**：快速定位零件
- **零件图片**：直观识别零件（已配置178张图片）
- **队伍编号**：记录队伍信息

### 💾 数据管理
- **自动保存**：浏览器本地存储，刷新不丢失
- **导出数据**：保存为JSON文件备份
- **导入数据**：从JSON文件恢复数据
- **清空功能**：可清空当前套件或重置所有数据

### 🖨️ 打印功能
- **智能打印**：只打印已使用的零件
- **紧凑布局**：3列网格，最大化利用纸张
- **单行统计**：使用统计显示在一行
- **自动隐藏**：未使用的零件和分类自动隐藏

## 🚀 快速开始

### 在线使用
访问：https://botball-check.itccc.app

### 本地使用
1. 下载项目文件
2. 双击打开 `index.html`
3. 开始使用！

无需安装任何软件，直接在浏览器中运行。

## 📖 使用指南

### 1. 输入队伍编号
在顶部输入框中输入你的队伍编号。

### 2. 选择套件
点击顶部标签切换不同的套件（电子/金属/乐高）。

### 3. 记录使用情况
- 使用 ➕➖ 按钮快速增减数量
- 或直接在输入框中输入数字
- 数据自动保存

### 4. 检查超规
- 顶部显示超规零件数量
- 超规零件卡片会变红色

### 5. 打印报告
点击 🖨️ 打印按钮，生成紧凑的使用报告。

### 6. 备份数据
- 点击 💾 导出数据保存为文件
- 使用 📥 导入数据恢复之前的记录
